## Tech Stack
* Clean Architecture with MVVM and DI with Dagger2
* Modern UI with Jetpack Compose
* Local Database for Caching: Room and SQLite
* Remote API: Retrofit
* Reactive Programming: Flow and Coroutine
* Other: Offline Connection Check, Caching
